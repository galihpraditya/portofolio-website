Salary Prediction Using Machine Learning Models

This project aimed to analyze how different regression models perform in salary prediction. Using Python’s robust libraries, I evaluated these models to identify the most accurate one. Random Forest emerged as the best-performing model, offering excellent prediction accuracy with the lowest MSE gap and the highest R² score.

Customer Churn Prediction

This project focuses on predicting customer churn in the telecom industry using machine learning. Customer churn, or the rate at which customers stop using a service, is a critical metric for businesses. By predicting churn, companies can take proactive measures to retain customers and improve customer satisfaction.
